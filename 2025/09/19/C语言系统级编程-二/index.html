3.几个注意事项：

  • Q typeof(T[N])等价于Q typeof(T)[N]，此时应该归类为Q T，注意和Q T[N](数组类型)的区别

  • 多个限定符语义：

    • 不同的限定符：限定符顺序与语义无关，例如const volatile int等价于volatile const int
    • 相同的限定符：等价于一个限定符[某些编译器会弹waring]

  • C23提供的函数typeof_unqual(T)可以获得T类型的非限定类型

  • _Atomc(T) 和 _Atomic T是两种类型，不是语法糖

    • _Atomc(T) ：T派生的原子类型，T不能是数组类型、函数类型、限定类型、原子类型
    • _Atomic T：T派生的原子限定类型，T不能是数组类型、函数类型

  • 限定类型的语法整体性：

    • Q T在语法上是一个整体

    • T Q在语法上是一个整体

    • T* Q在语法上不是一个整体

二.对象相关

1.对象的属性

  • 对象类型[Object Type]，也就是T，当然一个对象也可能没有对象类型
  • 对象名称[Object Name]，也就是O，有名称的对象被称为具名对象，反之称为匿名对象
  • 大小[Size]，对象占用内存的连续字节的个数
  • 地址[Adress]和对齐要求[Alignment]
  • 对象值[Object Value]和对象表示[Object Representation]，二者通过对象类型相互映射，其中对象表示就是0/1串

2.分配对象的方法

分类：对象定义、String Literal、Compound Literal、内存管理函数

a.对象定义

  • 对象定义的基本形式：

    • 内存中分配一个类型T的对象
    • 声明一个对象表示符O，让这个O和上述分配的T类型对象关联起来
    • 用initializer(初始化器)初始化这个对象的对象表示，initializer如果没有，则不构成对象定义
    • Storage-class specifiers指定对象存储周期

  • 不同对象类型的对象声明：

    建议一次只声明一个对象，因为一次定义多个可能会出现难以注意的问题，例如：

  • initializer(初始化器)用来设置对象的对象表示，说白了就是这个对象的对象值(Object Value)

  • 标量类型[scalar type] 和 非标量类型[non-scalar type]

    • 常见的标量类型：整数类型、浮点数类型、字符类型、布尔类型、枚举类型
    • 常见的非标量类型：数组、列表、字符串、结构体、字典、指针、容器、函数、方法

    标量类型存储单一不可分割的值，而非标量类型存储多个值或复杂结构

  • 标量类型对象及非标量对象的initializer

    • 标量类型对象的initializer

      • {}，空初始化列表，表示将对象的值设置为缺省值
      • 除逗号表达式之外的任意表达式exp或{exp}，意味着该exp求值之后的右值用来初始化对象表示

    • 数组类型对象的initializer

      • {}，空初始化列表
      • {sub-initializer1, sub-initializer2, … sub-initializern}

      给定数组类型T[N]，对于不同的情况：

      • N不存在，不能用第一种{}进行初始化，第二种情况下N就是数组长度
      • 当数组是变长数组时，只能用{}初始化
      • 普通数组，用{}初始化则元素递归被设置为缺省值，第二种情况下若n<N则剩下的值设置为缺省值

      一个例子：int a[2][3] = {1,2,3,4}

  • 对象表示与对象的表示值示例：

    显然a，b，c的对象的表示值不一样，但是在内存中表示它们的0/1串是一样的，即对象表示一致，还记得我们在上一篇文章中提到的对象表示和对象值之间的关系吗：假设有两个对象，他们的对象类型一样，

    • 若对象表示一样，则对象的值一定一样
    • 若对象的值一样，则对象表示不一定一样

    这里是否出现了矛盾？不是的，这两处的对象表示是一致的(内存中连续的0/1串)，但需要区分例子中的使用的术语是对象的表示值[Value]，而下述关系中使用的术语是对象值[Object Value]，对象的表示值与具体的对象类型有关，对后续表达式求值和右值至关重要

  • 对象的逻辑属性：对象的表示值 和 表示值类型

    • 非数组对象类型：<Object Value, typeof_unqual(Object Type)>，对象的表示值就是对象值，表示值类型是对象类型的非限定类型
    • 数组对象类型：<第一个元素对象的地址，元素类型对应的指针类型>，数组对象第一个元素对象的地址就是整个对象的地址，表示值类型是元素类型的指针类型，数组对象的对象表示不影响数组对象的表示值

b.String Literal字符串字面量

  • C语言定义的4种对象存储周期，决定了对象的有效期：

    • static

    • automatic

    • allocated

    • thread

      其中String Literal具有静态存储周期static，即有效期覆盖整个程序运期间，在程序运行之前其值会被初始化

      注意：C语言并没有全局变量和局部变量的概念（链接属性linkage和存储周期storage duration更重要）

  • 五种编码前缀用于指定字符串的字符编码格式或类型属性：

    • 无编码：以空字符（\0）结尾的 char 类型数组
    • u8：以空字符结尾的 char 类型数组
    • u：以空字符结尾的 char16_t 类型数组
    • U：以空字符结尾的 char32_t 类型数组
    • L：以空宽字符（L'\0'）结尾的 wchar_t 类型数组
